# Drifloon: Canadian Weather Data Download Toolkit

Downloads Canadian weather station data.

## Main Functions

- [`download_station_month()`](https://joeyfrost-xenopoulos.github.io/Drifloon/reference/download_station_month.md):
  Download one month of hourly data for a station.

- [`download_station_by_name()`](https://joeyfrost-xenopoulos.github.io/Drifloon/reference/download_station_by_name.md):
  Primary user entry for station downloads.

- [`download_station_province()`](https://joeyfrost-xenopoulos.github.io/Drifloon/reference/download_station_province.md):
  Download all stations in a province.

- [`download_all_station()`](https://joeyfrost-xenopoulos.github.io/Drifloon/reference/download_all_station.md):
  Download all stations in metadata.

- [`load_metadata()`](https://joeyfrost-xenopoulos.github.io/Drifloon/reference/load_metadata.md):
  Load packaged station metadata.

- [`sync_metadata()`](https://joeyfrost-xenopoulos.github.io/Drifloon/reference/sync_metadata.md):
  Sync metadata from official station inventory.

- [`download_metadata()`](https://joeyfrost-xenopoulos.github.io/Drifloon/reference/download_metadata.md):
  Copy packaged metadata `.rds` file to disk.
